Semantic HTML Reference

Main Tags

Tag Name Description
<header> Header Represents introductory content or navigation links for a section or page
<nav> Navigation Defines a section with navigation links.
<main> Main Section Represents the dominant content of the <body> of a document.
<section> Section Represents a standalone section of content that shares a common theme.
<article> Article Represents independent, self-contained content like blog posts or news items.
<aside> Aside Represents content indirectly related to the main content, like a sidebar.
<footer> Footer Defines a footer for a section or the entire page (e.g., copyright info).
<figure> Figure Encapsulates media content with optional caption.
<figcaption> Figure Caption Provides a caption for the <figure> element.

Common Attributes Used in Semantic Tags

Attribute Purpose Example
id Provides a unique identifier for an element. <section id="about">
class Assigns one or more class names for styling or scripting. <article class="news-item">
style Applies inline CSS styles. <footer style="color: gray;>
